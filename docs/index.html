Project Overview

This project is a comprehensive educational platform that leverages Retrieval-Augmented Generation (RAG) to help students and educators interact with PDF materials in innovative ways. The system processes uploaded PDFs, extracts and indexes content using vector embeddings, and enables users to chat with the material, generate quizzes, summaries, and flashcards.

The standout feature is an adaptive competitive quiz system that uses Q-Learning and Thompson Sampling algorithms to dynamically adjust question difficulty based on user performance, creating a personalized learning experience.

Technical Highlights

Prompt Engineering

  • Systematic prompting strategies with question-type detection
  • Dynamic prompt generation based on query type
  • Context management with token limits
  • Response post-processing for better formatting

RAG Implementation

  • Vector storage and retrieval with Pinecone
  • Hybrid chunking (semantic + page boundaries)
  • Effective ranking and filtering mechanisms
  • Namespace-based document isolation

Adaptive Learning

  • Thompson Sampling for exploration-exploitation
  • Real-time difficulty adjustment
  • Reward-based performance tracking

System Design

  • RESTful API design
  • Session-based document isolation
