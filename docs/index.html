Research Summary

Abstract

This research project presents a comprehensive educational platform that leverages Retrieval-Augmented Generation (RAG) and advanced prompt engineering to transform traditional PDF-based learning materials into interactive, AI-powered educational experiences. The system implements a sophisticated RAG pipeline using LlamaIndex and Pinecone vector database, combined with OpenAI's GPT-4 Turbo for intelligent content generation and question answering.

A key innovation is the implementation of an adaptive competitive quiz system that employs reinforcement learning algorithms (Q-Learning and Thompson Sampling) to dynamically adjust question difficulty based on real-time student performance. This creates a personalized learning trajectory that optimizes for both engagement and knowledge retention.

Key Contributions

  • Systematic Prompt Engineering: Seven-category question classification with dynamic prompt generation
  • Hybrid RAG Implementation: Novel chunking strategy combining semantic and page-based segmentation
  • Adaptive Learning Integration: Dual-algorithm approach (Q-Learning + Thompson Sampling) for difficulty optimization
  • Comprehensive Evaluation: LLM-based semantic evaluation for short-answer questions achieving 85% agreement with human experts
